# Portafolio Profesional

Practica guiada de desarrollo y versionamiento para la asignatura Desarrollo Web Integral.

## Enlaces de entrega

- Repositorio: https://github.com/EmilianoSP1/portafolio-profesional
- Historial de commits: https://github.com/EmilianoSP1/portafolio-profesional/commits/main
- Sitio en GitHub Pages: https://emilianosp1.github.io/portafolio-profesional/

## Evidencia de versionamiento

El historial incluye la configuracion inicial del portafolio, la incorporacion de estilos, una simulacion de cambio remoto, un cambio local simultaneo y el commit de resolucion del conflicto.

## Nota de despliegue

El sitio esta preparado para GitHub Pages. Si la URL publica muestra 404, se debe habilitar Pages en el repositorio desde Settings > Pages y seleccionar GitHub Actions o la rama main como fuente de publicacion.
